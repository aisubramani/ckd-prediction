💰 About This AI Medical Insurance Cost Prediction Tool

This AI-powered web application predicts a person's expected medical insurance cost using five key health and lifestyle factors — Age, Sex, BMI, Children, and Smoker status.

📊 Dataset Overview

The dataset contains 1338 records with 6 features:

FeatureDescription
AgeAge of the individual (years)
SexMale or Female
BMIBody Mass Index — a measure of body fat
ChildrenNumber of children covered by insurance
SmokerWhether the individual is a smoker (Yes/No)
ChargesMedical insurance cost in USD

🤖 Model Comparison

  • 📈 Linear Regression: MAE = 4213.48 | RMSE = 5829.17 | R² = 0.78
  • 📈 Ridge Regression: MAE = 4225.74 | RMSE = 5832.60 | R² = 0.78
  • 📈 Lasso Regression: MAE = 4214.20 | RMSE = 5829.46 | R² = 0.78
  • 🌲 Random Forest (Tuned): MAE = 2492.93 | RMSE = 4448.39 | R² = 0.87 ✅
  • ⚡ AdaBoost (Tuned): MAE = 3213.30 | RMSE = 4806.50 | R² = 0.85

Among all, the Random Forest Regression Model achieved the best prediction accuracy of 87%, making it the core model for this AI application.

🧠 Technical Stack

  • ⚙️ Framework: Python, Django, Scikit-learn, Pandas, NumPy
  • 📂 Model: Random Forest Regressor (GridSearchCV tuned)
  • 📊 Dataset: 1338 records × 6 columns
